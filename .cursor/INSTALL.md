# Install /dg for Cursor

Cursor rules are project-level. Download the rules file to the current project:

1. Create the directory: `mkdir -p .cursor/rules`
2. Download `https://raw.githubusercontent.com/v1r3n/dinesh-gilfoyle/main/cursor/dg.mdc` to `.cursor/rules/dg.mdc`

## Verify

Tell the user to invoke `@dg` in Cursor chat to verify the rule loads.
