# Install /dg for Codex CLI

Download these three files to `~/.codex/skills/dg/`:

1. Create the directory: `mkdir -p ~/.codex/skills/dg`
2. Download `https://raw.githubusercontent.com/v1r3n/dinesh-gilfoyle/main/dg/SKILL.md` to `~/.codex/skills/dg/SKILL.md`
3. Download `https://raw.githubusercontent.com/v1r3n/dinesh-gilfoyle/main/dg/gilfoyle-agent.md` to `~/.codex/skills/dg/gilfoyle-agent.md`
4. Download `https://raw.githubusercontent.com/v1r3n/dinesh-gilfoyle/main/dg/dinesh-agent.md` to `~/.codex/skills/dg/dinesh-agent.md`

## Verify

Tell the user to start a new Codex session and type `$dg` to verify the skill loads.
