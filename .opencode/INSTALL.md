# Install /dg for OpenCode

Download these three files to `~/.config/opencode/skills/dg/`:

1. Create the directory: `mkdir -p ~/.config/opencode/skills/dg`
2. Download `https://raw.githubusercontent.com/v1r3n/dinesh-gilfoyle/main/dg/SKILL.md` to `~/.config/opencode/skills/dg/SKILL.md`
3. Download `https://raw.githubusercontent.com/v1r3n/dinesh-gilfoyle/main/dg/gilfoyle-agent.md` to `~/.config/opencode/skills/dg/gilfoyle-agent.md`
4. Download `https://raw.githubusercontent.com/v1r3n/dinesh-gilfoyle/main/dg/dinesh-agent.md` to `~/.config/opencode/skills/dg/dinesh-agent.md`

## Verify

Tell the user to start a new OpenCode session and type `/dg` to verify the skill loads.
