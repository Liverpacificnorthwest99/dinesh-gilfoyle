# Install /dg for Windsurf

Download the rules file to Windsurf's global rules directory:

1. Create the directory: `mkdir -p ~/.codeium/windsurf/rules`
2. Download `https://raw.githubusercontent.com/v1r3n/dinesh-gilfoyle/main/windsurf/dg.md` to `~/.codeium/windsurf/rules/dg.md`

## Verify

Tell the user to ask for a "/dg review" in Cascade to verify the rule loads.
